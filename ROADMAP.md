# Roadmap

本文件列出 Multica Agent Workflow Template 的未来发展方向。优先级和排期可能根据社区反馈调整。

---

## 短期（1-2 个月）

- [ ] **GitHub Pages 文档站上线** — mdBook 构建 + GitHub Actions 自动部署
- [ ] **英文版文档** — `docs/en/` 目录，覆盖核心文档的英文翻译
- [ ] **更多 issue 描述模板** — 覆盖代码修复、论文写作、资料整理等场景
- [ ] **CONTRIBUTING.md** — 社区贡献指南

## 中期（3-6 个月）

- [ ] **更多 agent 结构案例** — 2-agent 极简版、按领域拆分版（纯写作/纯编程）
- [ ] **视频教程** — 从安装到第一个工作流的实操演示
- [ ] **自动化测试脚本** — 验证 skill router 路由准确性和 CLI operator 派单正确性
- [ ] **多语言 skill 路由** — 支持英文和日文的 skill 推荐

## 长期（6 个月以上）

- [ ] **工作流性能基准** — 不同模型/agent 数量下的响应时间和 token 消耗对比
- [ ] **社区工作流市场** — 用户可提交和分享自己的 agent 配置
- [ ] **与 Multica 官方深度集成** — 一键导入模板到新 workspace
