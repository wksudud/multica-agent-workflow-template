# 02 — 设计 Agent 结构

> 给 AI 的提示词，用于根据用户需求设计 agent 结构。

---

根据以下用户信息，设计适合的 Multica agent 结构：

## 用户信息

- 模型列表：<填入模型名称和能力>
- 预算和额度限制：<填入>
- 主要任务类型：<写作/编程/混合>
- 已安装 skills：<填入 skill 列表>
- agent 数量偏好：<少/适中/精细分工>
- 是否需要 manager：<是/否/不确定>

## 设计要求

1. 根据用户的实际模型和 skill 设计，不要照搬固定模板
2. 给出 agent 数量建议和理由
3. 为每个 agent 设计：
   - 名称（可用中英文）
   - 定位（一句话职责）
   - 推荐模型
   - 适合任务和不适任务
   - 推荐开启的 skills
4. 如果建议使用 manager agent，给出 manager 的完整指令草稿
5. 如果不需要 manager，说明替代方案

## 输出格式

请用表格和短文本输出，方便用户复制到 Multica agent 配置中。
