# 01 — 和 AI 讨论你的需求

> 复制以下内容，发送给你信任的 AI（或 Multica agent），开始讨论 agent 设计。

---

请帮我设计我的 Multica agent 工作流。先回答以下问题：

1. 你有哪些模型？（列出名称和能力评级）
2. 每个模型的能力如何？（强 / 中 / 经济）
3. 哪些模型价格较高？
4. 哪些模型有额度限制？
5. 你主要做写作、编程、研究还是混合任务？
6. 你安装了哪些 skill？（可以用 `multica agent list --output json` 查看）
7. 你希望 agent 少一点还是分工细一点？
8. 你是否需要一个总调度 agent（manager）来管理其他 agent？
9. 哪些类型的任务必须用最强模型？
10. 哪些类型的任务可以用便宜模型？

请根据我的回答，帮我设计 agent 的数量、命名、职责和 skill 分配方案。

注意事项：
- 不要假设我必须创建 7 个 agent
- 不要假设我必须区分写作和编程
- 根据我的实际情况推荐，不要照搬别人的配置
